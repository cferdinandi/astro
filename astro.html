How to use Astro Navbar Left Plus

1. Include Astro on your site.

<link rel="stylesheet" href="astro-navbar-left-plus.css">
<script src="astro-plus.js"></script>

For the "plus" versions of Astro, don't forget to include the astro-plus.js file.

2. Add the markup to your HTML.

<nav class="nav-wrap">
    <a class="logo" href="#">My Brand</a>
    <a class="nav-toggle" data-target="#nav-menu" href="#">Menu ▼</a>
    <div class="nav-collapse" id="nav-menu">
        <ul class="nav">
            <li><a href="#">Home</a></li>
            <li><a href="#">About</a></li>
        </ul>
    </div>
</nav>

Make sure that the data-target value on the .nav-toggle element matches the ID of the .nav-collapse section.

3. Add a feature test.

Astro "plus" versions use modern JavaScript API's that aren't supported by older browsers. Add this short snippet of code in the <head> of your HTML document to feature test:

<!-- Feature Test -->
<script>
    if ( 'querySelector' in document && 'addEventListener' in window ) {
        document.documentElement.className += 'js';
    }
</script>

For browsers that support modern JavaScript API's, this snippet adds a .js class to the <html> element, which activates the "plus" styling. Older and less capable browsers get the basic version.

4. Pat yourself on the back.

Seriously, you're done. Nice work! Modify as needed.
